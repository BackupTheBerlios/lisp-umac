This page was originally for the umac macro, but now i have a superior macro, denest. There are also added some other (reader)macros, like scope and unlisp.

Denest

Denest is a macro to de-nest code. Although the real thing also has some stuff to help with accumulating and to make good use of keywords, its basic form is:

(defmacro denest (&rest args) (if (null (cdr args)) (car args) `(,@(car args) (denest ,@(cdr args)))))

That simple, and if used properly, it is easy to execute back and forth in your mind. Given the right macros to input, it also seems to address the problem that other macros, like iterate, loop, bind try to solve in a more complicated fashion.

Metabang-bind, and let-star could somewhat do this too, but the makers didn't realize that this is (imo) what it comes down to; you are limited in what form it is entered, have to explicitly ask for it. (And let-star doesn't expose its define-binder, i forgot whether metabang-bind did.

I also mentioned it in this forum.

Usage

You can load it using either simply (load "denest.lisp") (Well the path to there too), or (asdf:oos 'asdf:load-op :denest) There are examples in use-rest/play-denest.lisp, and there is documentation.

Getting it

 Possibly to your convenience, i put the code on the website. Otherwise, if you have git, it is just git clone git://git.berlios.de/lisp-umac.
