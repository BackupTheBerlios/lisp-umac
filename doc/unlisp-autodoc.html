UNLISP - Not so lispy constructs.


[Macro]
binary &rest code => result


Fancy binary notation. (NOTE: much more inefficient then it could be, on plus side, all compile-time.)


[Function]
fancy what &optional how => result


Disable/enable unlisps fancy stuff, t enables default, others have other behaviors.


[Macro]
scope-fancy declaration* statement* => result


Adds fancy operators to define variables/functions to scope.
