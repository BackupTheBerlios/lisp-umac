Scope

Scope is a single macro that allows you to define variables/functions within a scope rather then valid in a body. Instead of:

(flet ((fun1 (arg1)...) (fun2 (arg2)...))
(let ((var1 (fun1 x)))
(flet ((fun-depending-on-that (blabla) ..))
..body..)))

You can write:

(scope
(funs (fun1 (arg1) ...)
(fun2 (arg2) ...))
(var var1 (fun1 x))
(fun fun-depending-on-that (blabla) ..)
..body..)

However, i found out that having to write more can often lead you to think of better ways to do things, so i am not using it.
